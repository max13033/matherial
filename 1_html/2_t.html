Тег < th> предназначен для создания одной ячейки таблицы, которая обозначается как заголовочная. Текст в такой ячейке отображается браузером обычно жирным шрифтом и выравнивается по центру. Тег < th> должен размещаться внутри контейнера < tr>, который в свою очередь располагается внутри тега < table>. Элемент < thead> предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы. Допустимо использовать не более одного элемента < thead> в пределах одной таблицы, и он должен идти в исходном коде сразу после тега < table>. < title> Определяет заголовок документа. Элемент < itle> не является частью документа и не показывается напрямую на веб-странице. Тег < tr> служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки может задаваться с помощью тега < th> или < td>.

Контейнер < tt> отображает текст моноширинным текстом.
